## 구현 결정사항

- `serverExternalPackages: ['dockerode', 'docker-modem', 'ssh2', 'better-sqlite3']` — Turbopack이 네이티브 모듈을 번들링하지 않도록 설정 (없으면 빌드 실패)
- `projects`, `settings` 페이지에 `export const dynamic = 'force-dynamic'` 추가 — SQLite 읽기라 정적 생성 부적합
- `lib/db-browser-stub.ts` 추가 — 클라이언트 번들에 better-sqlite3 포함 방지용

## 문제 및 해결

| 문제 | 해결 |
|---|---|
| Turbopack build error: ssh2 non-ECMAScript asset | `serverExternalPackages`에 ssh2, dockerode 추가 |
| TypeScript: `unknown[]` 타입 오류 (db.all()) | 반환 타입에 `as Type[]` 캐스팅 |
| NFT warning: process.cwd() 추적 | `/*turbopackIgnore: true*/` 주석 추가 |
| `Write` 도구 오류: 파일을 먼저 읽지 않음 | 기존 파일 수정 전 `Read` 필수 |
